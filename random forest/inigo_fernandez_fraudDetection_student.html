    Using only the training data, create an anomaly detection model. You should also choose an error metric adequate for the problem, and tune the model parameters in order to optimize this error.

    def CV_fnc(folds, dataset, target, IF):
    f1,precision,recall = [],[],[]
    for i, fold in enumerate(folds):
        X_train= dataset.loc[list(fold[0])].drop(target, axis=1)
        y_train= dataset.loc[list(fold[0])].loc[:,target]
        X_validation= dataset.loc[list(fold[1])].drop(target, axis=1)
        y_validation= dataset.loc[list(fold[1])].loc[:,target]
        IF.fit(X_train)
        y_validation_pred= IF.predict(X_validation)
        f1.append(f1_score(y_validation, y_validation_pred, pos_label=-1, average="binary"))
        precision.append(precision_score(y_validation, y_validation_pred, pos_label=-1, average="binary"))
        recall.append(recall_score(y_validation, y_validation_pred, pos_label=-1, average="binary"))
        
    f1= np.mean(f1)
    precision= np.mean(precision)
    recall= np.mean(recall)
    return (f1,precision,recall)

    Comprobamos qu mtrica nos arroja una mejor confusion matrix.
    Una confussion matrix adecuada es tal que el nmero de False Positive (FP) sea el menor posible sin descuidar el False Negative (FN) que sera el lado conservador.
    Tiene que ser un trade-off entre FP (prioritario) y FN.
    Por tanto, comprobamos los modelos que mejor mtrica han obtenido en el data train para cada una de ellas y calculamos su mariz de confusin.
    Si ninguna de ellas nos satifaciera, podramos creamos una mtrica que combinara ambas.

    precision: Positive Predictive Value, $\frac{TP}{TP+FP}$.
    Si $TP >> FP \to precision = 1$
    Si $FP >> TP \to precision = 0$

    recall: True Positive Rate, $\frac{TP}{TP+FN}$.
    Si $TP >> FN \to recall = 1$
    Si $FN >> TP \to recall = 0$

    F1: True Positive Rate, $\frac{2precisionrecall}{precision + recall}$
    Si alguna de las 2 mtricas se hiciera 0 (peor caso) F1 sera 0 $\to$ Cuanto mayor F1, mejor.

    Conclusin

    El objetivo es detectar los fraudes (TP) y que no haya fraudes que se clasifiquen mal (FP), como un dato normal no-fraude.
    Por tanto, la mtrica que minimiza el FP es la que nos arroja el mejor modelo. En este caso, dicha mtrica es recall con solo 35 fraudes no detectados.
    La cuestin es si se puede minimar el FP a cualquier precio. Los FN de la mtrica recall son 326 muy por encima de las otras dos mtricas. Ms del 300% en f1 y ms del 1.000% para precission.
    La mtrica precission es el polo opuesto a recall. Optimiza el FN sin prestar atencin al FP, obtiendo un FP inaceptable y muy mejorable.
    Una idea ms apropiada podra ser pensar en un trade-off entre FP y FN dando prioridad obviamente al FP del tal manera que no perjudicramos a clientes que por error se les considerara fraude sin serlo (FN).
    En esta lnea se encuentra la mtrica f1, que es un equilibrio entre FP y FN priorizando FP.

    def trainscore(Xtrain, ytrain, Xtest, ytest, estimator):
    """Trains an estimator and returns score over a test set"""
    estimator.fit(Xtrain, ytrain)
    return (estimator.score(Xtest, ytest) * 100, estimator.best_estimator_, estimator.cv_results_, estimator.best_params_)

def tune(Xtrain, ytrain, Xtest, ytest, estimator, param_dist, n_iter_search, n_jobs):
    """Performs hyperparameter tuning over an estimator"""
    try:
        search = RandomizedSearchCV(estimator, param_distributions=param_dist, n_iter=n_iter_search, n_jobs=n_jobs, scoring= make_scorer(f1_score, greater_is_better = True, average='binary'))
        return trainscore(Xtrain, ytrain, Xtest, ytest, search)
    except ValueError:
        search = GridSearchCV(estimator, param_grid=param_dist, n_jobs=n_jobs, scoring= make_scorer(f1_score, greater_is_better = True, average='binary'))
        return trainscore(Xtrain, ytrain, Xtest, ytest, search)

    def threshold_fnc(x, threshold):
    if x>threshold:
        return -1
    else:
        return 1

def model_fit_CV_fnc(folds, dataset, target, model, threshold):
    for i, fold in enumerate(folds):
        X_train= dataset.loc[list(fold[0])].drop(target, axis=1).values
        y_train= dataset.loc[list(fold[0])].loc[:,target].values
        X_validation= dataset.loc[list(fold[1])].drop(target, axis=1).values
        y_validation= dataset.loc[list(fold[1])].loc[:,target].values
        model.fit(X_train,y_train)
        y_validation_pred_prob= list(map(lambda x: threshold_fnc(x,threshold), model.predict_proba(X_validation)[:,0]))
        try:
            score.append(f1_score(y_validation, y_validation_pred_prob, pos_label=-1, average="binary"))
        except:
            score = f1_score(y_validation, y_validation_pred_prob, pos_label=-1, average="binary")
    return np.mean(score)
